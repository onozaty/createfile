# createfile

The createfile can create files of a specified size.

## Usage

```
$ createfile -t abc -s 1GB -o 1gb.txt
```

or 

```
$ createfile -b 616263 -s 1GB -o 1gb.txt
```

The arguments are as follows.

```
Usage of createfile:
  -b string
        binary pattern
  -h    Help
  -o string
        output file path
  -s string
        size
  -t string
        text pattern
```
